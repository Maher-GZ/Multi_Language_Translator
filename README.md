
# Multi_Language_Translator

![Multi Language Translator](Eng-Man.png)

## Overview

**Multi_Language_Translator** is a web application that leverages the Llama 3.1 8B model to perform translations between multiple languages, including English, German, French, and Hindi. The application is built using Streamlit, providing an intuitive and user-friendly interface for real-time translation.

## Features

- **Multi-language Support:** Translate text between English, German, French, and Hindi.
- **State-of-the-art Model:** Powered by the Llama 3.1 8B model with float16 precision, ensuring high-quality translations.
- **Streamlit Interface:** Clean, simple, and interactive web app for seamless user experience.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/Multi_Language_Translator.git
   cd Multi_Language_Translator
   ```

2. **Install the required dependencies:**

   You can install the required Python packages using pip:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit app:**

   ```bash
   streamlit run app.py
   ```

4. **Access the web app:**

   Once the server is running, you can access the web app at `http://localhost:8501` in your web browser.

## Usage

- **Input:** Enter the text you want to translate and select the source and target languages from the dropdown menu.
- **Output:** The translated text will be displayed instantly.
